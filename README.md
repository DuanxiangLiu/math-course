# 交互式初中数学课程平台

AI 驱动的交互式初中数学学习平台，基于国家课程标准开发，从七年级数学开始。

## 🎯 项目愿景

- **建构主义学习**：让学生通过操作、探索、发现来构建数学理解
- **即时反馈**：答题后立即给出详细解析和鼓励
- **进度跟踪**：记录学习进度，支持断点续学
- **AI 全程辅助**：从课程制作到内容优化，AI 全程参与

## 🚀 快速开始

### 启动开发服务器

```bash
./scripts/preview.sh
```

然后访问：http://localhost:6000

详细说明见 [docs/getting-started/QUICK_START_GUIDE.md](docs/getting-started/QUICK_START_GUIDE.md)
