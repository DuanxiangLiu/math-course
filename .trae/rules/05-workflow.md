# 工作流程

## 课程制作流程

### 1. 准备阶段
1. 查看 `textbooks/markdown/` 目录下是否有相关课本内容
2. 参考 `chapters/chapter1.html` 作为模板
3. 查看 `config.json` 了解课程结构
4. 确定章节编号和标题

### 2. 创建阶段
1. 在 `chapters/` 目录下创建新文件，命名为 `chapter{N}.html`
2. 复制 `chapter1.html` 的基本结构
3. 根据课本内容填充各个小节
4. 添加适当的交互组件
5. 确保数学内容的准确性

### 3. 测试阶段
1. 启动本地服务器：`python3 -m http.server 8080`
2. 在浏览器中预览新页面
3. 测试所有交互功能
4. 检查响应式布局
5. 验证数学内容的正确性

### 4. 完善阶段
1. 根据测试结果调整
2. 优化交互体验
3. 添加更多例题和练习
4. 确保进度跟踪正常工作
